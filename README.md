# 🛒  Projeto market list.py

**Descrição Geral:**
Um script interativo em Python desenvolvido para criar e gerenciar listas de compras via terminal, focando em usabilidade e controle de fluxo.

###  Funcionalidades Principais
* **Interatividade:** Sistema de pergunta inicial (Y/N) para engajar o usuário.
* **Loop Infinito Controlado:** Permite adicionar itens ilimitados até que o comando `STOP` seja acionado.
* **Padronização de Dados:** Uso de `.upper()` para garantir que o sistema entenda o comando de parada, independente de como foi digitado.

### Conceitos Técnicos Aplicados
* **Coleções:** Manipulação de listas com o método `.append()`.
* **Estruturas de Repetição:** Implementação de `while True` com interrupção por `break`.
* **Lógica Condicional:** Filtros `if/else` para direcionar o fluxo do programa.
* **Sanitização de Strings:** Tratamento de entradas para aumentar a confiabilidade do software.

###  Objetivo de Engenharia
Demonstrar o domínio de **Lógica de Programação**, manipulação de coleções de dados e criação de interfaces simples e funcionais no 3º semestre de Engenharia de Software.
